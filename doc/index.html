Eneboo

Eneboo es un fork de AbanQ 2.4, una aplicación orientada al desarrollo rápido de aplicaciones empresariales basadas fuertemente en base de datos bajo la liciencia GPLv2. Fue creado para poder introducir correcciones al programa y darle un aspecto más colaborativo al proyecto. Entre otras cosas, ofrece un sistema de compilación algo más sencillo y no hace uso de módulos firmados binarios.

Está ideado principalmente para trabajar sobre PostgreSQL como base de datos, y, aunque permite otros controladores (como MySQL o SQLite), el único recomendado es PostgreSQL.
